# Search in Sorted Array

## Problem statement

Given a sorted array of integers, arr, and a target value, target, return the target's index if it exists in the
array or -1 if it doesn't.

## Example 1

### Input

arr = [-2, 0, 3, 4, 7, 9, 11]
target = 3

### Output

Output: 2

## Example 2: 

### Input

arr = [-2, 0, 3, 4, 7, 9, 11]
target = 2

### Output

Output: -1
