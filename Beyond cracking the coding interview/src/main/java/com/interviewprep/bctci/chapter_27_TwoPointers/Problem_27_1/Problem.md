# PALINDROME CHECK

## Problem statement

Given a string s, return whether s is a palindrome. A palindrome is a string that reads the same forward and backward.

## Constraints

- The length of s is at most 10^6 
- s consists of lowercase English letters

## Example 1

### Input
s = "level"

### Output
true


## Example 2

### Input
s = "testeretset"

### Output
true

## Example 3

### Input
s = "hell0"

### Output

false
