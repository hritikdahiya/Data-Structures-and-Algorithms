# ARRAY REVERSAL

## Problem statement

Reverse an array of letters, arr, in place using O(1) extra space.

## Constraints

- 0 ≤ arr.length ≤ 10^5
- arr contains only letters

## Example 1

### Input

arr = ['h', 'e', 'l', 'l', 'o']

### Output

['o', 'l', 'l', 'e', 'h']

## Example 2

### Input

arr = ['a']

### Output

['a']

## Example 3

### Input

arr = []

### Output

[]
